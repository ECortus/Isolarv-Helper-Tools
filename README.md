# Custom-Tools

Custom tools/scripts and etc. for easier work.

## Dependencies

Install dependencies packages by insert rows into project `manifest.json`:

```
"com.unity.addressables" : "2.7.4",
"com.unity.addressables.android" : "1.0.7",
```

Dependencies included in project:

```
Extenject: https://github.com/modesttree/Zenject.git,
UniTask: https://github.com/Cysharp/UniTask.git,
In-game Debug Console: https://github.com/yasirkula/UnityIngameDebugConsole.git
```

## To install
Insert package into project file `manifest.json`:
``` 
"com.isolarv.custom-tools": "https://github.com/ECortus/Custom-Tools.git?path=Assets/Plugins",
```
