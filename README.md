# Custom-Tools

Custom tools/scripts and etc. for easier work.

## Dependencies

Install dependencies packages by insert rows into project `manifest.json`:

```
"com.unity.addressables" : "2.7.4",
"com.unity.addressables.android" : "1.0.7",
"com.cysharp.unitask": "https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask",
"com.svermeulen.extenject": "https://github.com/modesttree/Zenject.git?path=UnityProject/Assets/Plugins/Zenject",
"com.yasirkula.ingamedebugconsole": "https://github.com/yasirkula/UnityIngameDebugConsole.git",
```

## To install
Insert package into project file `manifest.json`:
``` 
"com.isolarv.custom-tools": "https://github.com/ECortus/Custom-Tools.git?path=Assets/Plugins",
```

### Game-Dev-Utils
Different scripts for similar work in code.

``` 
"com.isolarv.game-dev-utils": "https://github.com/ECortus/Game-Dev-Utils.git?path=Assets/Plugins/GameDevUtils",
```

### Caching-Objects-Tool
Tool to caching scriptable objects in Resources folder.

``` 
"com.isolarv.caching-objects-tool": "https://github.com/ECortus/Caching-Objects-Tool.git?path=Assets/Plugins/CachingObjectsTool",
```

### Localization-Module
Common localization module for Unity.

```
"com.isolarv.localization-module": "https://github.com/ECortus/Localization-Module.git?path=Assets/Plugins/LocalizationModule", 
```

### Game-Save-Kit
Tool to save & load game on Unity.

``` 
"com.isolarv.game-save-kit": "https://github.com/ECortus/Game-Save-Kit.git?path=Assets/Plugins/GameSaveKit",
```
